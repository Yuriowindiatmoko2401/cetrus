# Example Images

This directory contains example images for testing the DATSR Streamlit application.

## Naming Convention

- **LR images**: Include "lr" or "input" in the filename
- **Reference images**: Include "ref" or "reference" in the filename

## Adding Examples

To add example images for testing:

1. Place low-resolution images with names containing "lr" or "input"
2. Place reference images with names containing "ref" or "reference"
3. Supported formats: PNG, JPG, JPEG, BMP, TIFF, WEBP

Example structure:
```
assets/example_images/
├── lr_example1.png
├── lr_example2.jpg
├── ref_example1.png
└── ref_example2.jpg
```

These images will automatically appear in the "Try Example Images" section of the web interface.
